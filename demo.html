jquery-readme demo

Download

Basic Usage

	$('#test1').readmore();

Toggle Text

	$('#test2').readmore({ moreText: 'More!!!', lessText: 'Less!!!'});

Length

	$('#test3').readmore({ chars: 100 });

Duration

	$('#test4').readmore({ duration: 0 });

Newline

	$('#test5').readmore({ newline: true });	
	
	$('#test6').readmore({ newline: "<hr>" });

Reload

	$('#test7').readmore();

Click the button to toggle 'chars' option
