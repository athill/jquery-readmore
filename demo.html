jquery-readme demo

Basic Usage

	$('#test1').readmore();

Toggle Text

	$('#test2').readmore({ moreText: 'More!!!', lessText: 'Less!!!'});

Length

	$('#test3').readmore({ chars: 100 });

Speed

	$('#test4').readmore({ speed: 0 });

Newline

	$('#test5').readmore({ newline: true });

Prefix

	$('#test6').readmore({ prefix: 'test' });
