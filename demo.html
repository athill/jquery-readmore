jquery-readme demo

Download

Basic Usage

	$('#basics').readmore();

Toggle Text

	$('#toggle-text').readmore({ moreText: 'More!!!', lessText: 'Less!!!'});

Length (chars)

	$('#chars').readmore({ chars: 100 });

Duration

	$('#duration').readmore({ duration: 0 });

Newline

	$('#newline-true').readmore({ newline: true });	
	
	$('#newline-string').readmore({ newline: "<hr>" });

Reload

	$('#reload').readmore();

Click the button to toggle 'chars' option
